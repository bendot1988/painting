# Painting Website

Astro-based website project for a painting business.

## Prerequisites

- Node.js 18+
- npm

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

- `src/` - Astro pages, layouts, and components
- `public/` - Static assets served as-is
- `assets-research/` - Source/reference image assets

## Deployment

Any static hosting platform that supports Astro output can host this project.
